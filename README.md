## :question: Como usar

Para executar essa aplicação você precisará das seguintes ferramentas

- Yarn ou NPM
- Node >10
- Git

```bash
# Clone esse repositório
$ git clone https://github.com/joaovitors1g/RocketShoes

# Acesse a pasta recem criada
$ cd RocketShoes

# Instale as dependências
$ npm i

# Execute o app 🎉🎉
$ npm start
```
